# 走势图拖动与悬停信息 - 任务规划

## 1. 需求概述

1. 走势线缩放后支持鼠标拖动平移（Pan）
2. 鼠标悬停到走势线上时，显示十字准线和信息浮窗，包含：日期、净值、涨跌幅、累计收益、持有收益

## 2. 需求澄清记录

### Q1：拖动平移的触发方式
> **问题**：拖动平移的触发方式如何？
> - 鼠标按住拖动（不需要额外按键）
> - Alt+鼠标拖动
>
> **用户回答**：鼠标按住拖动（不需要额外按键）

### Q2：悬停信息展示的精度
> **问题**：悬停信息浮窗的值如何确定？
> - 精确到鼠标位置对应的数据点（根据 X 坐标找最近的数据点索引）
> - 仅在有数据点的位置显示
>
> **用户回答**：精确到鼠标位置对应的数据点（根据 X 坐标找最近的数据点索引）

### Q3：信息浮窗中累计收益和持有收益的数据来源
> **问题**：累计收益和持有收益的值是从哪里计算？
> - 利用已有的 `calcDailyProfits` 结果（累计收益=每日持仓市值-净投入）
> - 从交易记录实时计算
>
> **用户回答**：利用已有的 `calcDailyProfits` 结果

## 3. 涉及组件

| 组件 | 需要添加的功能 |
|------|--------------|
| `NavChart` | 缩放后鼠标拖动平移、悬停十字线+信息浮窗（日期/净值/涨跌/累计收益/持有收益） |
| `ProfitChart` | 缩放后鼠标拖动平移 |

## 4. 任务列表

### 任务 1：NavChart 拖动平移

| 属性 | 值 |
|------|-----|
| **描述** | 缩放后（zoomRange != null），鼠标按住 SVG 空白区域拖动可以平移可见范围。需要追踪 mousedown/mousemove/mouseup 事件，根据拖动距离调整 zoomRange 的 start/end 值 |
| **涉及文件** | `src/components/chart/NavChart.tsx` |
| **验收标准** | 放大后按住鼠标拖动，图表随拖动方向平移，x 轴标签同步更新 |

### 任务 2：NavChart 悬停信息浮窗

| 属性 | 值 |
|------|-----|
| **描述** | 鼠标在 SVG 上移动时，根据鼠标 X 坐标找到最近的数据点索引，绘制垂直十字线，并在浮窗中显示：日期、净值、涨跌幅、累计收益、持有收益。需要的额外数据：`transactions`（已有）、`summary`（已有）用于计算每日收益数据 |
| **涉及文件** | `src/components/chart/NavChart.tsx`、`src/components/RightPanel.tsx` |
| **验收标准** | 鼠标悬停在净值走势线上时，显示十字线和信息浮窗，浮窗内容随鼠标位置更新 |

### 任务 3：ProfitChart 拖动平移

| 属性 | 值 |
|------|-----|
| **描述** | 同任务 1，在 ProfitChart 中添加缩放后的鼠标拖动平移功能 |
| **涉及文件** | `src/components/chart/ProfitChart.tsx` |
| **验收标准** | 收益走势图放大后支持鼠标拖动平移 |

## 5. 执行顺序

```
任务 1 (NavChart 拖动) → 任务 2 (NavChart 悬停) → 任务 3 (ProfitChart 拖动)
     ↘ 并行可独立开发
```

- **任务 1 和 任务 2** 都修改 NavChart，有代码冲突，需要串行
- **任务 3** 与 任务 1 逻辑相同但修改不同文件，可以独立开发
