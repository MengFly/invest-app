# 收益与成本计算优化 - 任务规划

## 1. 需求概述

在投资详情页面中，区分并展示**持仓收益**与**累计收益**两种收益计算方式，以及**持仓成本线**与**累计成本线**两种成本计算方式。所有指标项支持悬浮 Tooltip 显示计算公式。

## 2. 需求澄清记录

### Q1：收益计算方式确认
> **问题**：持仓收益和累计收益的公式是否如下？
> - 持仓收益 = 持有市值 - 持仓成本（仅当前持有份额的加权平均成本）
> - 累计收益 = 持有市值 + 已卖出金额 - 总投入金额（包含已实现盈亏）
>
> **用户回答**：是的，完全正确

### Q2：成本线计算方式确认
> **问题**：两种成本线公式是否如下？
> - 持仓成本线 = 总买入金额 / 总买入份额（加权平均买入价）
> - 累计成本线 = (总买入金额 - 总卖出金额) / 持有份额（净投入/持有份额）
>
> **用户回答**：是的，完全正确

### Q3：展示位置确认
> **问题**：这些指标希望显示在哪里？
> - 顶部卡片 + 指标区域
>
> **用户回答**：两处都显示

### Q4：弹窗样式确认
> **问题**：点击弹出计算说明希望用哪种方式？
> - Tooltip 悬浮提示
>
> **用户回答**：Tooltip 悬浮提示

## 3. 计算公式说明

### 收益指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **累计收益** | `持有市值 - 净投入` = `holdAmount - totalInvested` | 已实现盈亏 + 未实现盈亏，当前已有 |
| **累计收益率** | `累计收益 / 净投入` | 当前已有（totalProfitRate） |
| **持仓收益** | `持有市值 - 持仓成本` = `holdAmount - avgCost × holdShares` | 仅当前持有份额的未实现盈亏 |
| **持仓收益率** | `持仓收益 / 持仓成本` | 基于加权平均成本的收益率 |

### 成本指标

| 指标 | 公式 | 说明 |
|------|------|------|
| **持仓成本线** | `总买入金额 / 总买入份额` | 加权平均买入单价（仅买入） |
| **累计成本线** | `净投入 / 持有份额` | 扣除卖出回收后的净成本单价 |
